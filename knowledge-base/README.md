# Knowledge Base Documents

Amazon Bedrock Knowledge Base에 업로드하는 코드 스타일 및 보안 원문입니다.

- `pep8.txt`: Python PEP 8
- `owasp-top10.txt`: OWASP Top 10
- `airbnb-style.txt`, `javascript_airbnb.txt`: JavaScript Airbnb 스타일
- `java_google.txt`: Google Java 스타일
- `golang_effective.txt`: Effective Go

실제 Knowledge Base ID는 `CODEBUDDY_KB_ID` 환경변수로 설정합니다.
