# Learning Examples

이 폴더는 1~6장 실습 중 최종 프로젝트의 발전 과정을 보여주는 대표 코드만 보존합니다.

- `bedrock/`: Bedrock 기본 호출과 프롬프트 기반 코드 분석
- `rag/`: Knowledge Base 문서 업로드, 검색, 출처 기반 리뷰
- `security/`: OWASP 문서 기반 보안 검사
- `legacy-github-tool/`: 통합 Tool 이전의 단독 GitHub PR Tool
- `demo-review/`: PR #9 자동 리뷰·수정·재리뷰 시연 코드

이 코드는 학습 참고용이며 최종 서버리스 배포에는 포함되지 않습니다. 실제 배포 코드는 `src/`, `lambda/`, `cloudformation/`을 사용합니다.
